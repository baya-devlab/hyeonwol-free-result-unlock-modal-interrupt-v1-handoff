# 현월 시행착오 / 인사이트 Evidence

## 시도한 퍼널 가설

* 카카오 공유 기반 해금으로 referral과 유료 전환을 동시에 만들 수 있다는 가설.
* 공유와 990원 직접 열람을 나눈 unlock modal로 공유 병목을 완화한다는 가설.
* 무료 결과페이지의 시각 보상과 블러 전문 섹션으로 전문 열람 의향을 높인다는 가설.
* 990원 속도화 해독 후 13,800원 심층 개안으로 객단가를 올린다는 가설.

## 시도한 광고 가설

* 도화살/첫인상/겉도화와 속도화/위험한 매력/타인이 보는 나의 차이 등 여러 메시지 축을 테스트했습니다.
* Core 구매 방어 소재와 신규 소재 screening을 분리했습니다.
* Pixel mismatch와 track beacon 문제를 수정한 뒤 표준 이벤트 기반 분석을 복구했습니다.

## 실제 관찰된 데이터

* 총 광고비: 726,775원
* 총 매출: 436,620원
* gross ROAS: 0.601
* Meta ViewContent: 2,020
* Meta InitiateCheckout: 220
* Meta Purchase: 144
* Admin order truth: 990원 218건, 13,800원 16건

## 실패한 가설

* paid acquisition 단독으로 단기 수익성을 맞춘다는 가설은 현재 데이터에서 실패했습니다.
* 990원 구매자가 충분한 비율로 13,800원으로 이어진다는 가설은 아직 강하게 입증되지 않았습니다.
* 신규 소재 screening만으로 ROAS를 빠르게 반전시키는 가설도 확인되지 않았습니다.

## 유효했던 신호

* 990원 구매는 반복적으로 발생했습니다.
* unlock modal과 무료 결과 UX 개편 후 전문 열람 의향 이벤트는 확인됐습니다.
* Pixel-fix 후 Meta 표준 이벤트 수신은 복구됐습니다.
* 일부 공유/속도화 메시지 소재는 상대적으로 구매 방어 신호가 있었습니다.

## 다음 상품에 재활용 가능한 자산

* 무료 결과 → 선택형 해금 modal → 블러 전문 섹션 구조.
* Core 소재와 신규 소재를 분리해 screening하는 운영 구조.
* Meta 표준 이벤트와 Admin order truth를 함께 보는 분석 방식.
* 도화/첫인상/관계 해석형 카피 자산.

## 주의해야 할 리스크

* 저가 상품은 광고 구매가 발생해도 단가 구조상 수익성이 낮습니다.
* 고가 업셀은 가격 설득과 타이밍이 약하면 전체 ROAS를 끌어올리지 못합니다.
* 배포/계측 변경이 많았기 때문에 기간별 cohort를 나누지 않으면 해석이 왜곡됩니다.
